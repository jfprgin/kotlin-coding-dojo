# Coding Dojo

> Start with Exercise 1 and work your way forward. It's perfectly fine if you don't finish everything -- we care about your approach, not just the result.

**Setup:**

```bash
# Build the project (ensures everything compiles)
./gradlew build

# Run all tests
./gradlew test

# Run tests for a single exercise
./gradlew test --tests "exercise1.*"
./gradlew test --tests "exercise2.*"
./gradlew test --tests "exercise3.*"
```
